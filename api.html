looted.link

Below are some endpoints to retreive items created by users of looted.link. It should be noted that all of these items were created by users of the app and do not reflect the my personal views. A vast majority of the items were created by Reddit users so it's safe to say a few items may be offensive to some people.



Lists of Created Items

GET

/api/items/slot/<slot>

Get created items by slot.
head neck shoulder chest wrist hands legs feet weapon

GET

/api/items/quality/<quality>

Get created items by quality.
uncommon rare epic

GET

/api/items/all

Get all created items.
all


Individual Created Items

/api/item/<id>

Get created items by slot.
